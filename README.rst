使い方
==================================================

| $ hg clone https://bitbucket.org/mittostar/dotfiles
| $ cd dotfiles
| $ sh setup.sh

| 必要そうなことはsetup.shがやってくれます
| 個別にやりたい場合は自分でシンボリックリンク張ってください
